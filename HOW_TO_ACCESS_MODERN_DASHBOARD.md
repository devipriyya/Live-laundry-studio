# How to Access the Modern Admin Dashboard

## 🚀 Quick Start Guide

### Step 1: Start the Application

```bash
# Navigate to the backend folder
cd backend
npm start

# In a new terminal, navigate to the frontend folder
cd frontend
npm run dev
```

### Step 2: Access the Login Page

Open your browser and navigate to:
```
http://localhost:5173
```

### Step 3: Login as Admin

On the landing page, click on the **Login/Register** button or navigate to the admin login.

Use admin credentials:
- **Email**: admin@fabricspa.com (or your admin email)
- **Password**: Your admin password

### Step 4: Navigate to Modern Dashboard

After successful login, you'll be automatically redirected to:
```
http://localhost:5173/admin-dashboard
```

This is the **NEW MODERN ADMIN DASHBOARD**!

## 🌙 Toggle Dark Mode

Click the **moon icon** 🌙 in the top right header to switch to dark mode.
Click the **sun icon** ☀️ to switch back to light mode.

## 🔔 Check Notifications

Click the **bell icon** in the top right to view notifications:
- New orders
- Low inventory alerts
- Payment confirmations
- System updates

## 🔍 Search Feature

Use the search bar in the header to search across:
- Orders
- Customers

## 🐛 Troubleshooting

### Dashboard Not Loading?
- Check if backend server is running (port 3000)
- Check if frontend server is running (port 5173)
- Verify admin credentials

### Styling Issues?
- Clear browser cache
- Hard reload (Ctrl+Shift+R or Cmd+Shift+R)
- Check browser console for errors

### Navigation Not Working?
- Ensure JavaScript is enabled
- Try a different browser
- Check for ad-blocker interference
